# 説明
- 共通仕様と個別仕様の両方に下記のテンプレートを適用する。
- 指定のない項目も空欄で列挙すること。
- 必要に応じて項目を追加して良い(項目名は漢字2字にし既存の項目名を優先して利用すること)。

**名称**：UIの呼称、
**種類**：UIの種類、
**用途**：目的、機能、
**形状**：形状、質感、角、枠線、
**寸法**：サイズ、長さ、高さ、幅、
**文字**：フォント、サイズ、太さ、行間、整列、
**色彩**：色全般、透明度、
**装飾**：エフェクト、シャドウ、グラデーション、
**配置**：位置、スペース、間隔、階層、関係、
**状態**：変化、遷移、
**挙動**：変形、移動、インタラクティブ、
