# カード追加ボタン (FAB)

新規カードを作成するためのフローティングボタン。

# デザイン

## カード追加ボタン
**名称**：カード追加ボタン
**種類**：フローティングアクションボタン
**用途**：新規カードの作成
**形状**： 12px の角丸正方形、 1px の紫色枠線 (  #9333eaff  )
**寸法**： 56px x 56px
**色彩**：背景色 白色 (  #ffffffff  )、アイコン色 紫色 (  #9333eaff  )
**装飾**：紫色のシャドウ (  #a855f74d  )
**配置**：
  - [右下] に固定
  - 右端から 32px
  - 下端からの距離：
    - フッター表示時： 104px
    - フッター非表示時： 32px
  - Z-Index: 100
**状態**：ホバー時拡大 (1.05倍)、アクティブ時縮小 (0.95倍)
**挙動**： 300ms のトランジション。リスト内のカード展開時は非表示。
