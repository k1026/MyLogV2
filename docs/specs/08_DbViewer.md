# 8. データ閲覧・管理 (DbViewer)
- アプリケーションのデータを格納しているデータベースの中身をユーザーが確認するためのビューアーを提供
- ライトテーマで指定がない限り他のダイアログとデザインを揃える

## 8.1 デザイン
- DB表示UIはダイアログで開く
- 特に指定がない場合はUIは黒色
### 8.1.0 閉じるボタン
- ダイアログ右上に閉じるボタンを表示
- ボタンは薄い紫色

### 8.1.1 ロゴ
- 中央上部に太文字の「Database」をロゴとして表示する
- ロゴの上部に薄い小さな文字で「LOCAL」を表示
- ロゴの左側にデータベースを表すアイコンを表示

### 8.1.2 操作ボタン
- 操作ボタンはロゴの下に横並びに配置 

- **操作ボタン共通デザイン**：操作ボタンは薄い枠線と文字で構成
  - タップされた場合はボタンの指定色で塗りつぶし文字を白抜きの反転デザインにする
  - 対象の処理が完了するまでボタンの色の反転状態を維持する

- **DeleteDBボタン**：左側に配置する赤色のボタン。データベースを削除する。
  - 操作時には確認ダイアログを表示する。

- **Appendボタン**：中央に配置する紫色のボタン。ファイルダイアログを開き指定したファイルをデータベースに**追加**する。
  - 追加処理の実行中はプログレスバーを表示する
  - 追加中は中止するボタンを表示し、中止ボタンを押すと追加処理を中止する
  - 追加完了時は完了を伝える表示と読み込んだデータとスキップしたデータの件数を表示する

- **Exportボタン**：右側に配置する紫のボタン。ファイルダイアログを開き、指定したフォルダにデータベースをファイル出力する。
  - 出力中はプログレスバーを表示する
  - 出力中は中止ボタンを表示し、中止ボタンを押すと出力処理を中止する
  - 出力完了時は完了を伝える表示と出力したデータの件数を表示する

### 8.1.3 リスト情報
- 操作ボタンの下部に表示されるデータベースの情報を表示するUI

- **アイテム数**：リストの左上に表示されるデータベース内のアイテム数。左詰めで配置。

- **ページ切り替えボタン**：リストに表示する情報を切り替えるボタンでリスト上部中央に表示
  - 紫色の「<」と「>」を使ったシンプルなデザイン。
  - ページの端に到達した場合は対応するボタンをグレーアウト。

- **ページ番号**：ページ切り替えボタンの間に配置するページ番号表示。
  - 背景などのないシンプルな表示で「(現在のページ)/(ページ総数)」の形式で表示する。
  - 現在のページ番号をクリックすると数値を直接入力できる(数値の増減ボタンなどは不要)

### 8.1.4 リスト表示
- **リスト**：データベース内の情報をレコード(セル)単位で閲覧するリスト
- **カラム**：列表示としてデータベース内のカラムを使用する
  - カラムの表示順は左から「ID」、「GPS」、「ATTRIBUTE」、「NAME」、「VALUE」、「REMOVE」の順に表示
- **レコード**：リスト内ではレコード単位で情報を表示する
  - リストのレコードはIDの新しい順に並べる
  - リストはレコードを100件ずつ表示し縦と横のスクロールを許可する。

## 8.2 起動時の動作
- DB表示ダイアログ起動時にはDBの先頭から100件ずつ順に読み込んで表示しユーザーが読み込まれたデータを閲覧できるようにする
  - 読み込み中は操作ボタンをグレーアウトして無効化する
    - ページ切り替えボタンは操作可能にする
