# フッター仕様

## レイアウト
- **[全体]** ：水平
  - **[左側]**：水平
    - 🔃 **ソート切り替え**：昇順・降順の切り替え
  - **[中央]**：水平
    - 🔍 **フィルタ**：フィルタダイアログの開閉・無効化
  - **[右側]**：水平
    - 📱 **ビュー切り替え**：リスト・グリッド表示の切り替え

## 共通仕様
**名称**：フッター / Footer
**種類**：容器 (Container / Fixed Bar)
**用途**：画面下部での主要操作ナビゲーションの提供
**形状**：矩形、上部境界線のみ(1px)
**寸法**：高さ 80px、最大幅 1280px (max-w-7xl)
**文字**：-
**色彩**：背景 白、境界線 ライトグレー (#e2e8f0)
**装飾**：上部シャドウ (微弱)
**配置**：画面下部固定、最前面 (z-index: 50)
**状態**：
- **表示 (Default)**：画面内に表示
- **非表示 (Hidden)**：画面外 (下方向) へスライド退避
**挙動**：
- **下方スクロール/入力中**：非表示状態へ遷移 (translate-y-full)
- **上方スクロール/フォーカスアウト**：表示状態へ遷移 (translate-y-0)

## ソート切り替え
**名称**：ソート切り替えボタン / SortButton
**種類**：切替ボタン
**用途**：カードリストの時系列ソート順の変更 (昇順 ⇔ 降順)
**形状**：アイコン背景：角丸 (12px)
**寸法**：アイコン領域：40px x 40px
**文字**：ラベル：10px、太字、大文字、トラッキング拡大
**色彩**：
- 通常：テキスト グレー (#94a3b8)、背景：透明
- ホバー：テキスト 濃いグレー、背景：薄いグレー (#f1f5f9)
- アクティブ (昇順時)：テキスト 紫 (#9333ea)、背景：薄紫 (#f3e8ff)
**装飾**：アイコンの180度回転遷移 (duration-300)
**配置**：左端に配置
**状態**：
- **降順 (初期)**：Newestラベル、通常スタイル
- **昇順**：Oldestラベル、アクティブスタイル (紫)
**挙動**：
- クリック時に昇順・降順をトグル
- **処理仕様**：全データ読み込み後はメモリ上で即時ソート
- **視覚連動**：処理中はヘッダーのDBアイコンを紫色に変更し「sorting」を表示

## フィルタ
**名称**：フィルタボタン / FilterButton
**種類**：アクションボタン / トグル
**用途**：フィルタダイアログの開閉、または適用中のフィルタの一時無効化
**形状**：アイコン背景：角丸 (12px)
**寸法**：アイコン領域：40px x 40px
**文字**：ラベル：10px、太字、大文字、トラッキング拡大
**色彩**：
- フィルタオフ：通常スタイル (グレー)
- フィルタオン：アクティブスタイル (紫)
- フィルタ無効：透過・グレースケール表示 (opacity-50 grayscale)
**装飾**：-
**配置**：中央に配置
**状態**：
- **オフ**：フィルタ未設定。タップでダイアログを開く。
- **オン**：フィルタ適用中。タップで「無効」へ切り替え。
- **無効**：フィルタ設定はあるが適用外。タップでダイアログを開く。
**挙動**：タップ時に状態に応じたアクションを実行

## ビュー切り替え
**名称**：ビュー切り替えボタン / ViewModeButton
**種類**：切替ボタン
**用途**：カードリストの表示形式変更 (リスト ⇔ グリッド)
**形状**：アイコン背景：角丸 (12px)
**寸法**：アイコン領域：40px x 40px
**文字**：ラベル：10px、太字、大文字、トラッキング拡大
**色彩**：
- リスト：通常スタイル (グレー)
- グリッド：アクティブスタイル (紫)
**装飾**：アイコンの切り替え
**配置**：右端に配置
**状態**：
- **リスト (初期)**：1列表示、Listラベル
- **グリッド**：2列表示、Gridラベル
**挙動**：
- 表示モードのみを即座に変更 (フィルタ/ソートの再計算は行わない)
