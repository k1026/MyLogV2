# ロジック

- **単位の統一**: フォントサイズはすべて `px` で定義する。
- **色彩の統一**: 色の値はすべて `16進数 (Hex)` 形式（例: ` #eb5635ff `）で定義し、前後にスペースを入れる。
- **フォント**: 基本は `font-sans` を使用し、アイコンは `Material Symbols Rounded` (標準 24px) を使用する。
- **動的変化**: 背景色（レア度）に応じてテキスト色を  #ffffffff  に反転させるロジックを保有する。
