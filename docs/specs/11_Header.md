# ヘッダー仕様

## レイアウト
- **[全体]**：水平
  - **[左側]**：垂直
    - 📍 位置情報設定アイコン
    - 📊 カード統計
      - カード枚数
      - セル数
  - **[中央]**：垂直
    - 🖼️ アプリロゴ
    - 🔢 バージョン番号
  - **[右側]**：水平
    - 🎲 ランダム抽出
    - 🗄️ データベースアイコン

## 共通仕様
**名称**：ヘッダー (Header)
**種類**：ナビゲーションコンポーネント
**用途**：ナビゲーション、ステータス、ブランド、アクションの提供
**形状**：全幅矩形、下端境界線
**寸法**：高さ 64px (h-16)、コンテンツ最大幅 1280px (max-w-7xl)、左右パディング 16px (px-4)
**文字**：Sans-serif (font-sans)
**色彩**：背景色は白、境界線はグレー (`border-slate-200`)
**装飾**：下部シャドウ (`shadow-sm`)
**配置**：画面最上部固定 (`fixed top-0`)、Zレイヤー 50 (`z-50`)
**状態**：表示/非表示 (スクロール連動・入力中連動)
**挙動**：上下スライドイン/アウト (`transition-transform duration-300`)、スクロール連動（下方スクロールで `-translate-y-full`、上方スクロールで `translate-y-0`）、入力操作連動（input/textareaフォーカス時に非表示）

## 個別仕様

### アプリロゴ
**名称**：アプリロゴ
**種類**：ブランド表示
**用途**：アプリケーションのアイデンティティ表示、クリックによる初期状態へのリセット
**形状**：矩形 (テキスト)
**寸法**：
**文字**：サイズ 20px (text-xl)、特太 (font-black)、字間 狭 (tracking-tight)
**色彩**：通常時 パープル (`text-purple-300`)、読込中 パープル (`text-purple-600`)
**装飾**：
**配置**：ヘッダー中央、垂直方向の上側
**状態**：DB読込中 (pulseアニメーション)
**挙動**：クリックでアプリのリセットを実行

### バージョン番号
**名称**：バージョン番号
**種類**：情報表示
**用途**：アプリケーションのバージョン確認
**形状**：矩形 (テキスト)
**寸法**：
**文字**：サイズ 12px、太字、Sans-serif
**色彩**：グレー (`text-slate-400`)
**装飾**：親要素（タイトルエリア）ホバー時に不透明度上昇
**配置**：ヘッダー中央、垂直方向の下側
**状態**：
**挙動**：

### 位置情報設定
**名称**：位置情報設定
**種類**：切替ボタン
**用途**：位置情報の取得・設定状態の表示と、取得ON/OFFの切り替え
**形状**：正円 (rounded-full)
**寸法**：アイコンサイズ 16px、全体 24px
**文字**：--
**色彩**：
- 背景色：なし
- 有効：アイコンの中を紫色 (`fill-purple-300`)、線は白色、
- エラー時：赤色 (`text-red-500`) で塗りつぶし (`fill-red-500`) 斜め線 (`MapPinOff`)
- 無効：薄い灰色
- 取得中：薄い灰色
**装飾**：なし
**配置**：ヘッダー左側、垂直方向の上側
**状態**：取得中/有効/無効/エラー
**挙動**：タップで位置情報取得の有効/無効切り替え

### カード統計
**名称**：カード統計
**種類**：ステータス表示
**用途**：現在カードリストに表示されているカード件数と含まれるセルの数を表示する
**形状**：矩形 (テキスト)
**寸法**：--
**文字**：サイズ 10px、太字、大文字、字間 広 (tracking-widest)
**色彩**：グレー (`text-slate-500`)
**装飾**：--
**配置**：
- ヘッダー左側、垂直方向の下側、
- カード枚数が上、セル数が下、左詰め
**状態**：--
**挙動**：--
