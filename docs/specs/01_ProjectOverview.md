# 1. プロジェクト概要
MyLogは、ユーザーが日々の記録（ログ）を簡単に作成・保存できる軽量なWebアプリケーションです。オフライン環境でも動作することを前提とし、すべてのデータをブラウザ内のデータベース（IndexedDB）に保存します。

## 1.1 技術スタック
- **言語**: HTML5, CSS3, JavaScript (ES6+)
- **フレームワーク**: なし (Vanilla JS)
- **ストレージ**: IndexedDB (ブラウザ内蔵データベース)
- **位置情報**: Geolocation API
- **動作環境**: モダンブラウザ（Chrome/Edge等）、オフライン対応
