# 14. 位置情報管理 (Location Service)
- アプリ起動時に位置情報の監視を開始します (`watchPosition`)。
- ヘッダーのアイコンでステータスを表示します。
  - **点滅**: 取得中
  - **実線**: 取得完了（有効）
  - **エラー**: 取得失敗

-**精度**：位置情報の精度は常に使用できる最高精度を採用すること

- **Cell生成時の記録**:
  - 新規Cell（Card, Time, Text, Task含む全て）の作成時に、その時点の最新の位置情報を `geo` 属性に記録します。
  - フォーマット: `"緯度 経度 高度"` (半角スペース区切り文字列)。
  - 例: `"35.6895 139.6917 10.5"` (高度が取得できない場合は `0` を記録します)
  - 位置情報が取得できていない場合、またはエラー時は `null` を記録します。
