# 4. セルUI (Cell UI)

セルUIは情報単位であるCellの表示・編集を行うUIである。

## 4.1 共通仕様 (Common Specifications)

共通デザインおよび挙動は以下の通り定義する。

**名称**：セル共通UI,
**用途**：各種データの表示・編集インターフェース,
**形状**：
- 基本：32px（3xl）の角丸、細い白色の枠線（border-white/80）
- Timeセル：角丸なし、枠線なし
**寸法**：
- 幅：親要素に対して100%（w-full）
- 最小高さ：5rem（Timeセルは自動調整）
**色彩**：
- 背景：rgba(255, 255, 255, 0.6) の半透明白色
- Timeセル：透明（transparent）
**装飾**：背景ぼかし（backdrop-blur-md）、影（shadow-sm）、500msの遷移エフェクト
**配置**：
- レイアウト：Flexbox（縦並び、gap-3）
- テキスト配置：左詰め
**状態**：
- フォーカス時：入力フィールドの下線が紫色 (`border-purple-500`) に変化
**挙動**：
- 自動保存：フォーカスが外れた場合や入力完了時にデータベースへ保存
- 操作禁止：`geo` および `remove` メンバはユーザーから直接操作・表示しない

## 4.2 セルタイプ別仕様 (Specs by Cell Type)

### 4.2.0 カードセル(Card Cell)
Attribute：`Card`
**Cardセルに関しては別途仕様書を用意するのでこの仕様書では実装を意識しなくて良い**

### 4.2.1 タイムセル (Time Cell)
**名称**：タイムセル (Time),
**種類**：テキストボタン,
**用途**：発生・記録日時の表示・編集,
**形状**：矩形（角丸・枠線なし）,
**寸法**：
- 高さ：コンテンツに応じて自動調整
- 文字サイズ：タイトル 14px、値 14px
- 横幅：コンテンツの文字列に揃える余白は作らない
**色彩**：
- 文字：白色（text-white）
- プレースホルダー：透明度40%の白
**装飾**：なし,
**配置**：
- タイトルと値は縦並び（間隔 2px）
- 日付と時刻は横並び（間隔 2px）
**状態**：
- フォーカスリングは非表示
**挙動**：
- 日時・時刻フィールドをクリックするとOS標準のピッカーを表示
- 入力値に応じた自動幅調整

### 4.2.2 テキストセル (Text Cell)
**名称**：テキストセル (Text),
**用途**：名前（タイトル）と値（メモ内容）の保存・編集,
**形状**：基本仕様に準拠,
**寸法**：
- 文字サイズ：タイトル 20px (Bold)、値 18px
**色彩**：
- 文字：タイトル slate-800、値 slate-700
- プレースホルダー：slate-400
**装飾**：基本仕様に準拠 (背景ぼかし等),
**配置**：
- タイトルと値を縦並び（間隔 3）
**状態**：
- 入力フィールドにフォーカスがある間、下線が紫色に強調
- キーワード一致時にテキストハイライトを表示
**挙動**：
- 値のテキストエリアは入力内容に応じて自動で高度が拡張される
- 未入力かつ非フォーカス時はフィールドを非表示にする（最小高さは維持）
- エンターキーでタイトルから値へフォーカス移動
- タイトル推定候補（Estimation）の表示・選択

### 4.2.3 タスクセル (Task Cell)
**名称**：タスクセル (Task),
**用途**：To-do管理、完了状態の管理,
**形状**：
- チェックボックス：角丸 2px
- セル本体：基本仕様に準拠
**寸法**：
- チェックボックス：28x28px (w-7 h-7)
- 文字サイズ：タイトル 20px (Bold)
**色彩**：
- チェックボックス：枠 purple-200、チェック時背景 purple-500
- 完了時：テキスト slate-400、打ち消し線 decoration-slate-300
**装飾**：チェック時にチェックマーク（✓）を表示,
**配置**：
- チェックボックスとタイトルを横並び（間隔 3）
**状態**：
- 完了状態（Checked）に応じてスタイルが変化
**挙動**：
- チェックボックスのクリックで完了状態をトグルし自動保存
- タイトル推定候補（Estimation）の表示・選択
