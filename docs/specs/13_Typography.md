# 13_Typography

本アプリケーションにおけるフォント、文字サイズ、カラーの仕様定義。

**セルUI**
    - セルUIの各種フォントサイズの定義
        - 各種セルの`name`は指定がない場合20pxを基本とする
            - `Time`セルの`name`は14px
        - 各種セル`value`は18px
        - セルの生成時刻表示は10px
        - プレースホルダーも上記に準拠

**カードUI**
    - カードUIのフォントサイズは指定がない限りセルUIの定義に準拠する

**ヘッダーUI**
    - フォントサイズの定義
        - アプリタイトルは25px
        - バージョン番号は10px
        - カードとセルの総数表示は12px

**フッターUI**

- **統一アイコン**: Header/Footer等のアイコンは、アプリケーション全体で統一されたデザインとカラーパレットを使用します。
