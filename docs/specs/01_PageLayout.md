# 1. メインページレイアウト

## 1.0 テーマ
- アプリ全体のテーマはモダンなデザインにする。
- 指定がない限りライトテーマを基本とする

## 1.1 構成
- 本アプリはメイン画面のみとしページ移動は行わない。
- メイン画面はヘッダー、メインコンテンツ、フッターの3つの領域で構成される。
- それ以外のUIはモーダルダイアログで表示する。

## 1.2 メインページレイアウト
- メインページには下記のUI要素を含みます。詳細は各仕様書を参照。
    - ヘッダー：各種機能を持ったアイコンを配置するヘッダー
    - Cardリスト：DBから読み込んだカード情報を一覧表示するリスト
        - Card追加ボタン：Cardリストに新しいカードを追加するフローティングボタン。
    - フッター：各種機能を持ったアイコンを配置するフッター
