# 実装計画: カードリスト内での展開挙動への修正

## 概要
カード追加・展開時に、他のカードを非表示にするのではなく、リスト内で展開しスクロール範囲を制限する挙動に変更する。

## 実施手順

### 1. CardList.tsx: レンダリング論理の緩和
- `visibleCards` の計算で `expandedCardId` による単一カードへの絞り込みを廃止。
- 展開開始時の `scrollTop` に基づき、その時に表示されていた範囲のカードを保持する。

### 2. CardList.tsx: スクロール範囲制限の実装
- カードが展開されている間、`onScroll` イベントでスクロール位置を制限。
- クランプ範囲: `[targetCardTop, targetCardBottom + 80px]`

### 3. Card.tsx: 高さ情報の伝播
- 内部の状態変化（セルの追加等）があった場合に、必要に応じてリスト側にリサイズを通知する。

### 4. テストと動作確認
- Vitest で展開中に他のカードが DOM に残っていることを確認。
- 実機で、展開されたカードの範囲外にスクロールできないことを確認。
